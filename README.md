# decks

Package decks provides a simple API for creating and customizing decks of cards in Go.

## Installation
```
go get github.com/preslavmihaylov/decks
```

## Quickstart
```
package main

import "github.com/preslavmihaylov/decks"

func main() {
  
}

```
